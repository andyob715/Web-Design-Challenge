Latitudinal Study of Weather

The purpose of this project was to explore how the weather changed as an observer moved closer and further away from the equator. This was achieved by completing the following steps:

  1. Generating random lat/long values via randrange function in python
  2. Using the citipy library to attempt to find the closest city to those random coordinates
    • Duplicates were expected so the number of coordinates retrieved far exceeded the goal value of 750
  3. Using the openweather API, the city names were plugged into the API call and if cities could be found they were returned
    • Weather features such as Temperature, Humidity, Wind Speed and Cloud Cover were returned through the API
  4. Each weather feature was plotted against latitude to validate whether any coorelation to the latitude appeared
